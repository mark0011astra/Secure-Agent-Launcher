# UI Theme Rule

UIを新規作成・改修するときは、以下のDark System Colorsを標準テーマとして使用する。

## Accent Colors (Dark)

- `blue`: `#0A84FF`
- `green`: `#30D158`
- `indigo`: `#5E5CE6`
- `orange`: `#FF9F0A`
- `pink`: `#FF375F`
- `purple`: `#BF5AF2`
- `red`: `#FF453A`
- `teal`: `#64D2FF`
- `yellow`: `#FFD60A`

## Gray Colors (Dark)

- `gray`: `#8E8E93`
- `gray2`: `#636366`
- `gray3`: `#48484A`
- `gray4`: `#3A3A3C`
- `gray5`: `#2C2C2E`
- `gray6`: `#1C1C1E`

## Application Rule

- 画面のベースは `gray6` / `gray5` / `gray4` を使う。
- 重要操作のみ accent color を使い、それ以外は gray 系で統一する。
- 無効状態は全体を gray 系へ寄せて、無効であることが一目で分かるようにする。
